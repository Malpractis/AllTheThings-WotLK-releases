# AllTheThings for WoW 3.3.5a - releases

Prebuilt releases of the [AllTheThings](https://github.com/ATTWoWAddon/AllTheThings)
collection addon adapted for WoW 3.3.5a (WotLK) private-server clients.

## Install (once)

1. Download **`ATTUpdater.exe`** from the [latest release](../../releases/latest).
2. Run it (SmartScreen may warn once: More info -> Run anyway).
3. Confirm your WoW folder when asked.

It installs the addon and keeps both the addon and itself up to date
automatically from then on. Open "AllTheThings Updater" from the Start Menu
any time to see status or force a check.

## Manual install

Download `AllTheThings-335-<version>.zip` from the latest release and extract
the `AllTheThings` folder into `<WoW>\Interface\AddOns\`.

Every release includes `SHA256SUMS`; verify with `sha256sum -c SHA256SUMS`.
